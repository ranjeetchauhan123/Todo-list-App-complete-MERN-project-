# 📝 Todo List App (Full Stack)

A full-featured **Todo List Application** built using **MERN stack** with authentication and authorization. This app supports complete CRUD operations along with secure user login/signup using JWT.

---

## 🚀 Features

* ✅ User Signup & Signin (JWT Authentication)
* 🔐 Authorization with Middleware
* 📝 Create, Read, Update, Delete (CRUD) Todos
* ☑️ Select All Todos
* 🗑️ Delete All Todos
* 🍪 Cookie-based Authentication
* 🌐 CORS Enabled
* ⚡ Fast and Responsive UI

---

## 🔐 Authentication Flow

* User registers via **Signup**
* User logs in via **Signin**
* JWT token is generated and stored in cookies
* Protected routes are secured using middleware
* Only authorized users can access their todos
